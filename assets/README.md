# Saleh — Assets

Bu qovluqda app-ın production üçün lazımi şəkilləri olmalıdır:

| Fayl | Ölçü | Format | Qeyd |
|---|---|---|---|
| `icon.png` | 1024×1024 | PNG | iOS + qaynaq icon |
| `splash.png` | 1284×2778 | PNG | iPhone Pro Max splash |
| `adaptive-icon.png` | 1024×1024 | PNG | Android adaptive (foreground) |
| `notification-icon.png` | 96×96 | PNG | Android notification (monochrome ağ) |
| `favicon.png` | 48×48 | PNG | web (opsional) |

## Faza 1 üçün

Expo sənə default placeholder assets verir — `npx create-expo-app` istifadə etsən, bu fayllar avtomatik yaranır. Skeleton-da yalnız `app.json` config-ə yazılıb, faktiki şəkillər sənin tərəfindən gəlməlidir.

## Icon önə təklif

- **Brand mark:** kalliqrafik `S` və ya minimalist mehrab arch
- **Rəng:** ağ üzərinə qızıl (`#D4A574`) və ya tünd mavi (`#0F1B2D`) üzərinə qızıl
- **Stil:** flat, gradient yox, naxış yox — sakit, ciddi
